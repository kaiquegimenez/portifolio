    HTML

    HTML é uma linguagem de marcação utilizada na construção de páginas Web, pois é essa a linguagem interpretada pelos navegadores. Com isso Utilizo O HTML para fazer a construção básica dos meus projetos.

    CSS

    CSS é uma linguagem de estilo utilizada para descrever a apresentação de um documento escrito em HTML. Com isso utilizo o CSS para estilizar as páginas dos meus projetos.

    Angular2+

    Angular é uma plataforma e framework para construção da interface de aplicações, a qual nos ajuda a criar Single-Page Applications com uma qualidade e produtividade surpeendente! Utilizo o Angular2+ em projetos maiores.

    Javascript

    JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web. Utilizo o Javascript para construção de páginas Web não estáticas.

    Vue.js

    Vue.js é um framework JavaScript de código-aberto, focado no desenvolvimento de interfaces de usuário e aplicativos de página única. Utilizo ele em grandes projetos que necessitam funcionar me multiplas plataformas

    Git e GitHub

    GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Utilizo para armazenamento e controle de versão dos meus projetos.
